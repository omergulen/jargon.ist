İki argüman alan fonksiyonlara `binary function` adı verilir. Örnek olarak;

```js
const add = (x, y) => x + y;

// add fonksiyonu binary bir fonksiyondur.
```

Ayrıca [binary operation](/binary-operation) jargonuna da bakılabilir.
