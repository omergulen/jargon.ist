Argüman _(argument)_, bir fonksiyona gönderilebilen ve fonksiyon tarafından işlenebilen değerdir. `parameter` ya da `parametre` olarak da kullanılır.

```js
// x, add fonksiyonun ilk, y ise ikinci argument'idir.
const add = (x, y) => x + y;
```
