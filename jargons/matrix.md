Matrixler, dizilerin çok boyutlu halleridir.

Örneğin, [Java](/java)'da;

```java
// kullanacağımız array in iki boyutlu belirtmek için [][] şeklinde kullandık.
int[][] myMatrix = { {0,1,2,3}, {3,2,1,0}, {3,5,6,1}, {3,8,3,4} };
```
